# 너와나 배포 방법

코드 변경 후 아래 순서대로 진행하세요.

---

## 1. 서버 (Railway)

GitHub에 푸시하면 Railway가 자동 배포합니다.

```bash
cd C:\Users\USER-PC\Desktop\랜챗
git add .
git commit -m "변경 내용 설명"
git push origin main
```

---

## 2. 앱 (Netlify) - 수동

### 1단계: 빌드

```bash
cd C:\Users\USER-PC\Desktop\랜챗\neowana
flutter build web --release --base-href /
```

### 2단계: 업로드

1. **파일 탐색기**에서 `C:\Users\USER-PC\Desktop\랜챗\neowana\build\web` 폴더 열기
2. **Netlify** → [app.netlify.com](https://app.netlify.com) → 사이트 선택
3. **Deploys** 탭 → **Drag and drop** 영역에 `web` 폴더 내용 끌어다 놓기
4. 배포 완료 대기 (1~2분)

### 3단계: 확인

- 사이트 접속 후 **Ctrl + Shift + R** (강력 새로고침)

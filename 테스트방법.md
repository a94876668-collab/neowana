# 너와나 테스트 방법

## 중요: 반드시 **서로 다른** 두 개의 연결이 필요합니다

---

## 방법 1: Chrome + Edge (가장 확실)

1. **서버 실행**
   ```
   cd 랜챗\server
   npm start
   ```

2. **터미널 1** - Chrome에서 앱 실행
   ```
   cd 랜챗\neowana
   flutter run -d chrome
   ```
   → Chrome 창이 열림

3. **주소창에서 URL 복사** (예: `http://localhost:52345`)

4. **Edge 브라우저를 열고** 주소창에 그대로 붙여넣기 후 접속

5. **Chrome**에서 "매칭 시작" 클릭

6. **Edge**에서 "매칭 시작" 클릭

7. 매칭되면 채팅 가능!

---

## 방법 2: Chrome 탭 2개

1. `flutter run -d chrome` 실행 후 Chrome에서 앱 열기
2. **주소창 URL 복사** (예: http://localhost:52345)
3. **새 탭**(Ctrl+T) 열고 주소창에 같은 URL 붙여넣기
4. 두 탭 각각에서 "매칭 시작" 클릭

---

## 방법 3: 터미널 2개에서 flutter run

1. **터미널 1**: `cd 랜챗\neowana` → `flutter run -d chrome`
2. **터미널 2**: `cd 랜챗\neowana` → `flutter run -d chrome`
3. Chrome 창이 2개 열리면 각각에서 "매칭 시작"

---

## 서버 로그 확인

서버 터미널에서 다음이 보여야 합니다:
- `[연결] user_1_xxx` - 첫 번째 사용자
- `[연결] user_2_xxx` - 두 번째 사용자
- `[매칭 완료] user_1_xxx <-> user_2_xxx`

`[연결 해제]`가 매칭 직후 바로 보이면 연결이 끊긴 것입니다.
